## Connections

- LED `-> PIC GPIO` through resistor; Button `-> GPIO` with pull-up; Sensor `-> ADC`. UART to PC via TTL-

## Diagram (ASCII)

```
 GPIO ---[330Ω]---|> LED --- GND
 Button -> GPIO (pull-up)
 Sensor -> ADC
 UART TX/RX -> USB-TTL
```
